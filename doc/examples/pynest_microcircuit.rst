PyNEST microcircuit
===================

This is a PyNEST implementation of the microcircuit model by Potjans and Diesmann.

.. toctree::
   :maxdepth: 1

   pynest_microcircuit_link

You can open files used directly here.

File structure
--------------

* :doc:`run_microcircuit.py <../_build/html/auto_examples/Potjans_2014/run_microcircuit.py>`: an example script to try out the microcircuit
* :doc:`network.py <../_build/html/auto_examples/Potjans_2014/network.py>`: the main Network class with functions to build and simulate the network
* :doc:`helpers.py <../_build/html/auto_examples/Potjans_2014/helpers.py>`: helper functions for network construction, simulation and evaluation
* :doc:`network_params.py <../_build/html/auto_examples/Potjans_2014/network_params.py>`: network and neuron parameters
* :doc:`stimulus_params.py <../_build/html/auto_examples/Potjans_2014/stimulus_params.py>`: parameters for optional external stimulation
* :doc:`sim_params.py <../_build/html/auto_examples/Potjans_2014/sim_params.py>`: simulation parameters
* `reference_data <https://github.com/nest/nest-simulator/tree/master/pynest/examples/Potjans_2014/reference_data>`_
