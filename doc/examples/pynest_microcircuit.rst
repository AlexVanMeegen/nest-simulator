PyNEST microcircuit
===================

Here is the PyNEST implementation of the microcircuit model by Potjans and Diesmann.

.. toctree::
   :maxdepth: 1

   pynest_microcircuit_link

You can open files used directly here.

File structure
--------------

* :download:`run_microcircuit.py <../pynest/examples/Potjans_2014/run_microcircuit.py>`: an example script to try out the microcircuit
* :download:`network.py <../pynest/examples/Potjans_2014/network.py>`: the main Network class with functions to build and simulate the network
* :download:`helpers.py <../pynest/examples/Potjans_2014/helpers.py>`: helper functions for network construction, simulation and evaluation
* :download:`network_params.py <../pynest/examples/Potjans_2014/network_params.py>`: network and neuron parameters
* :download:`stimulus_params.py <../pynest/examples/Potjans_2014/stimulus_params.py>`: parameters for optional external stimulation
* :download:`sim_params.py <../pynest/examples/Potjans_2014/sim_params.py>`: simulation parameters
